== About Commerce Cloud B2C Data Connector

Anypoint Connector for Salesforce Commerce Cloud B2C Data APIs is an operation-based connector, which means that when you add the connector to your flow, you need to configure a specific operation for the connector to perform.

Commerce Cloud B2C Data Connector enables the user to access the Salesforce Commerce Cloud B2C Data API by using certain configurations, and to then access the data, update the data, and delete the data, as required.

== Prerequisites

Before creating an app, you must have access to the Commerce Cloud B2C Data Connector target resource and
Anypoint Platform. You also need to understand how to create a Mule app using
Anypoint Design Center (Design Center) or Anypoint Studio (Studio). 

Further, you must be familiar with Mule runtime engine (Mule), Anypoint Connectors, Anypoint Studio Essentials, Mule flows, and Mule Global Elements.

To use Commerce Cloud B2C Data Connector, you need:

* Anypoint Studio version 7 and later
* Commerce Cloud B2C Data Connector
* Commerce cloud credentials such as client Id, Client Secret, Scopes to access Data API which the user needs to get from the Salesforce team.
